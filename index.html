Homebrew Physics Arcade

Built by @jayyvarmaa


Five mini games powered by one homebrew 2D physics engine.

This site was built just for fun experimentation to test how physics can work on a website. There is no roadmap beyond this prototype.

How To Play

A/D move, Space jump, S fast-fall. Orange platforms are bouncy. Touch yellow to finish.
